# 필드 타입 필수 여부 설명
1 Education str 필수 ['Graduation' 'PhD' 'Master' 'Basic' '2n Cycle']
2 Marital_Status str 필수 ['Single' 'Together' 'Married' 'Divorced' 'Widow' 'Alone' 'Absurd' 'YOLO']
3 Year_Birth int 선택 생년 ex) 1957
4 Income int 선택 수입 ex) 60000
5 Kidhome int 선택 ex) 1
6 Teenhome int 선택 ex) 0
예시:
http://{{ host }}/api/pred?Year_Birth=1957&Education=PhD&Marital_Status=Married&Income=80000.0&Kidhome=1&Teenhome=1
